# Java POO Gestión Inventario

Java project created to practice object-oriented programming concepts through a simple inventory management system.

## Objective
Practice core OOP concepts with a small but useful project.

## Concepts used
- Classes and objects
- Attributes and methods
- Encapsulation
- Constructors
- Lists
- Basic logic and validation

## Main idea
The system will allow:
- Register products
- Show product list
- Update stock
- Search products
- Calculate total inventory value

## Planned technologies
- Java
- Object-Oriented Programming
- GitHub
